# WW1-Airplane-Model-OpenGL
Modelling First World War Airplane with OpenGL

*  Compile File
```shell
g++ main.cpp camera.h -lglfw -lGL -lGLU -lglut -o main
```
*  Camera Movement

    **w**    : up
    
    **a**    : left
    
    **s**    : down
    
    **d**    : right
    
    **r**    : reset camera position
    
    **q**    : exit
